## Key Features

<CardGroup cols={2}>
  <Card title="8-bit CPU" icon="microchip" href="/architecture/cpu">
    8 general-purpose registers with flag system for zero, overflow, and underflow detection
  </Card>
  <Card title="64KB RAM" icon="memory" href="/architecture/memory">
    Full 64KB addressable memory space with memory-mapped I/O
  </Card>
  <Card title="Stack Operations" icon="layer-group" href="/architecture/stack">
    256-byte dedicated stack with push, pop, call, and return support
  </Card>
  <Card title="16x16 Display" icon="display" href="/architecture/display">
    Built-in graphical display with memory-mapped framebuffer
  </Card>
</CardGroup>

## Rich Instruction Set

Neodymium VM provides over 50 opcodes covering:

<CardGroup cols={3}>
  <Card title="Data Movement" icon="arrows-turn-right">
    MOV instructions for register, immediate, and memory operations
  </Card>
  <Card title="Arithmetic" icon="calculator">
    ADD, SUB, MUL, DIV, INC, DEC, PWR, SQRT, and MOD operations
  </Card>
  <Card title="Logic & Bitwise" icon="code-branch">
    AND, NOT, and comparison operations
  </Card>
  <Card title="Control Flow" icon="route">
    Conditional and unconditional jumps with flag-based branching
  </Card>
  <Card title="Stack & Subroutines" icon="brackets-curly">
    PUSH, POP, CALL, and RET for function calls and stack management
  </Card>
  <Card title="Memory Access" icon="database">
    STORE instructions for writing to RAM and memory-mapped devices
  </Card>
</CardGroup>

## Quick Example

Here's a simple program that adds two numbers:

```asm example.asm
; Load values into registers
MOV $a, #10    ; Load 10 into register $a
MOV $b, #20    ; Load 20 into register $b

; Add them
ADD $a, $b     ; $a = $a + $b (result: 30)

; Halt with result
HALT $a        ; Exit with code in $a
```

## System Requirements

<Note>
  Neodymium VM currently supports UNIX-based operating systems only (Linux, macOS). Windows support is planned for future releases.
</Note>

**Minimum Requirements:**
- 128 KB of RAM
- 1 CPU core
- 8 MB storage

**Build Dependencies:**
- C++17 or higher compiler
- CMake 3.31.6 or higher
- GLFW 3.4 or higher (for display output)
